# SIPERKASA Backend

Sistem Informasi Penunjukan Jaksa

## 🚀 Features
- JWT Authentication
- Role-based access (admin, kajari, jaksa)
- SPDP → auto Perkara
- P16 Assignment (multi jaksa)
- Pagination, Filter, Search, Sorting

## 🛠 Tech Stack
- Node.js
- Express
- Sequelize
- PostgreSQL

## 📌 Endpoint Example
GET /perkaras  
GET /perkaras?page=1&limit=5  
GET /perkaras?search=andi
